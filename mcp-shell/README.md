# MCP-SHELL v1.3.1

PowerShell command execution server for GitHub Copilot via Model Context Protocol.

## Features

- Base64 encoding (Exit Code 255 fix)
- Circuit Breaker pattern
- Smart Retries
- Watchdog (30s timeout)
- Adaptive Timeout Policy
- Error Classification (7 types)
- Concurrency Control (max 5 processes)

## Installation

Use the automatic installer: `MCP_SHELL_INSTALLER.ps1`
