### Troubleshooting

**Flow doesn't start:**
- Check System Status (Ollama, CORTEX running)
- Try `quick_status` first
- Check `logs/flows/` for errors

**Training fails:**
- Verify dataset path in flow JSON
- Check PULSE v1 status in TrainingPanel
- See training logs

**Git errors:**
- Run `git_check` to see blockers
- Fix repo state (commit changes, fetch remote)
- Try `smoke_test` again
