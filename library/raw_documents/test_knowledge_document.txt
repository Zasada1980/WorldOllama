## Содержание

Это тестовый файл содержит образец знаний, которые должны быть автоматически приняты системой `ingest_watcher.ps1` и перемещены в библиотеку.

### Принцип 10 ТРИЗ: Предварительное исполнение

**Формулировка:** Заранее выполнить требуемое действие (полностью или хотя бы частично).

**Применение в Data Tray:**
- Файл размещается в "лотке" заранее
- Система автоматически его обрабатывает
- Нет необходимости в ручном управлении

### Ожидаемое поведение

При запуске скрипта:
1. ✅ Файл должен быть обнаружен в `inputs/data_tray`
2. ✅ Имя должно быть санитизировано (пробелы → подчеркивания)
3. ✅ Кодировка проверена (UTF-8 ожидается)
4. ✅ Файл перемещен в `library/raw_documents`
5. ✅ Событие залогировано в `logs/ingestion.log`
