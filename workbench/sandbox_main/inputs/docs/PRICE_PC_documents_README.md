Эта папка содержит исходные текстовые файлы библиотеки знаний.

- **Общий размер:** ~1.5 GB (до индексации)

## Индексация

Файлы автоматически индексируются через LightRAG при обновлении репозитория.

**Процесс:**
1. Файл разбивается на чанки (10KB каждый)
2. Извлекаются сущности и связи
3. Строится граф знаний
4. Создаются векторные эмбеддинги

## Добавление новых файлов

```bash
# 1. Добавьте .txt файл в эту папку
# 2. Commit и push
git add documents/новый_файл.txt
git commit -m "docs: добавлен новый_файл"
git push

# 3. Индексация запустится автоматически (если настроен auto-sync)
```
